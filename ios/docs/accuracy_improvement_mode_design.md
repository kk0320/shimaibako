# 精度向上モード設計

## 目的

寝る前や充電中など端末状態が良い時に、OCR済み写真の再分類、未分類写真の軽量再判定、スクショ細分類、手動分類傾向の反映、検索インデックス整理を少しずつ進める。

3万枚規模でも端末が固まらないように、一度に処理する件数を絞る。元写真・元動画は保存、送信、変更しない。

## 初期実装

初期実装は手動実行中心にする。設定画面に「今すぐ少しだけ実行」ボタンを置き、1回あたり最大50件だけ処理する。

処理内容:

- 未分類写真の軽量分類再判定
- OCR済み写真のカテゴリ再判定
- スクショ細分類の再判定
- 手動分類傾向の反映
- 検索インデックス整理

初期実装では新規OCRや画像特徴量生成は行わない。画像本体を読み込まず、保存済みメタデータとOCR結果、手動分類傾向だけで再判定する。

## 設定

設定画面で次を扱う。

- 精度向上モード: ON/OFF
- 実行タイミング
  - 手動のみ
  - 夜間に自動実行を試みる
- 推奨時間帯: 23:00〜6:00
- iCloud取得
  - オフライン優先
  - iCloud取得を許可

説明文:

「夜間や充電中に、分類の精度を少しずつ高めます。元写真・元動画は外部送信せず、端末内で処理します。」

## 安全条件

実行前と処理中に端末状態を確認する。

- 充電中またはバッテリー50%以上
- 低電力モードOFF
- 発熱状態が通常またはやや高め
- 空き容量2GB以上推奨
- Wi-Fi推奨

中断条件:

- キャンセル操作
- 低電力モードON
- `thermalState` が `serious` または `critical`
- バッテリー50%未満かつ充電中ではない
- 空き容量1GB未満

中断時は完了済みの再判定だけ保存する。未処理分は次回以降に残す。中断そのものを写真単位の失敗として扱わない。

## 処理対象の選び方

読み込み済み写真から最大50件を選ぶ。

優先順:

1. 未分類写真
2. OCR済み写真
3. スクショ
4. その他の画像

手動分類がある写真では、手動分類を自動判定や分類傾向学習より優先する。

実行時に手動分類済み写真が対象に含まれた場合、その写真は自動再判定をスキップし、手動分類保護件数として記録する。これにより、ユーザーが直した分類を精度向上モードが上書きしない。

## 実行履歴

実機検証で確認しやすいように、設定画面に次を表示する。

- 開始日時
- 終了日時
- 実行結果
- 実行モード
- 対象件数
- 完了件数
- 失敗件数
- 中断件数
- 手動分類保護件数
- 中断理由

個別写真名、写真パス、OCR全文は表示しない。履歴は `UserDefaults` に保存し、アプリ再起動後も確認できるようにする。

## 精度向上データ削除

設定画面から「精度向上データを削除」を実行できる。確認ダイアログを必ず表示する。

削除対象:

- 分類傾向学習データ
- 将来の画像特徴量データ
- 精度向上モードの処理履歴

削除しないもの:

- 元写真・元動画
- OCR結果
- 手動分類
- 写真アプリ側の写真、アルバム、iCloud写真

ファイル削除はアプリ内のローカルデータ削除対象に限定する。写真アプリ内の元写真・元動画、PhotoKitアセット、OCR結果、手動分類を削除対象にしない。写真ライブラリ全体の安全方針は `ios/docs/photo_library_global_safety_policy.md` に記録する。

精度向上データ削除は、分類傾向学習データ、将来の画像特徴量データ枠、処理履歴だけを対象にする。検索インデックス整理や軽量再判定でも、OCR原文データと手動分類結果は保持する。手動分類が存在する写真は自動再判定より手動分類を優先し、精度向上モードの処理中は上書きせず保護件数として記録する。

実装上のファイル削除は `LocalDataDeletionTarget` で列挙されたアプリ内データだけに限定する。削除直前に `Application Support/ShimaiBako` 直下のファイルであることを確認し、元写真・元動画、PHAsset、写真アプリ側のアルバム、OCR結果、手動分類結果を削除対象にできないようにする。

## 画像特徴量の扱い

Vision Feature Printなど画像特徴量の本格処理は初期実装では行わない。

将来使う場合の方針:

- 花・植物、建物、芸術品、食べ物、動物などOCRで弱いカテゴリの補助に使う
- 未分類、手動分類済み、新規写真などに限定して試す
- 全3万枚を一気に処理しない
- 充電中、Wi-Fi、発熱なし、空き容量ありの場合だけ処理する
- 特徴量は端末内保存に限定する
- 元写真・元動画は保存・送信しない
- 精度向上データ削除で消せるようにする

## 軽量検証
