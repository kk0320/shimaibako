# 精度向上モード実機検証チェックリスト

## 目的

実機iPhoneで精度向上モードを安全に確認する。元写真・元動画、OCR結果、手動分類を壊さず、端末内の軽量な再判定と履歴表示だけを検証する。

## 前提

- 元写真・元動画は削除、移動、変更しない
- 写真アプリ内の元写真・元動画、アルバム、iCloud写真は削除、移動、変更しない
- OCR結果は削除しない
- 手動分類は自動再判定より優先する
- 精度向上データ削除は、分類傾向学習データ、将来特徴量枠、処理履歴だけを対象にする
- 個別写真名、写真パス、OCR全文は検証ログに書かない
- 空き容量不足テストは無理に実施しない
