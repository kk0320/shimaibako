# 写真アクセス権限の挙動

## 基本方針

しまい箱は写真を読み取り専用で扱う。写真本体を外部送信せず、削除、移動、リネーム、上書き、共有は行わない。

## 権限状態

### 未確認

写真アクセス前の説明画面を表示し、許可ボタンからシステムの写真アクセス許可へ進む。

### 許可

許可された写真ライブラリから直近100件を読み込み、グリッド、詳細、検索、OCRに利用する。

### 限定アクセス

選択中の写真のみ読み込む。設定画面には「選択中の写真のみ利用できます」と表示し、「写真の選択を変更」から標準の限定選択画面を開く。

限定アクセス中も、許可範囲内の写真だけでグリッド、検索、OCRを実行する。選択外の写真を直接参照しない。

### 拒否

写真一覧は空にし、設定アプリへの導線を表示する。拒否状態で写真読み込みやOCRを開始しない。

### 制限

端末や管理設定により写真アクセスが制限されている状態として説明する。写真一覧は空にし、アプリ内では安全に待機する。

## 設定画面に表示する情報

- 写真アクセス状態
- 読み込み上限
- OCR済み件数
- OCR未処理件数
- OCR失敗件数
- 外部送信: なし
- 保存先: 端末内
- 写真操作: 読み取り専用

## 実機確認で見ること

1. 未確認から許可へ進めること
2. 限定アクセスで選択した写真だけが表示されること
3. 「写真の選択を変更」で選択範囲を変えられること
4. 拒否状態でアプリが落ちないこと
5. 制限状態の説明が自然に表示されること
6. 設定画面の安全方針カードの表示順が維持されていること
