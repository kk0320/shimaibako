# iCloud写真の扱い

## 基本方針

しまい箱はPhotoKit経由で写真を読み取る。iCloud写真を取得する場合も、外部OCRサービスへ写真を送るのではなく、Apple/iCloud写真から端末へ必要な画像を取得する扱いになる。

## 設定

設定画面で次を選べる。

| モード | 挙動 |
| --- | --- |
| オフライン優先 | `PHImageRequestOptions.isNetworkAccessAllowed = false` とし、端末内にある画像を優先する |
| iCloud取得を許可 | `PHImageRequestOptions.isNetworkAccessAllowed = true` とし、OCR時に必要な画像を取得できる |

設定値は `UserDefaults` に保存する。

## OCR時の挙動

- オフライン優先では、iCloud上にあり端末に未取得の画像はOCRできない場合がある
- 取得できない場合はアプリを落とさず、OCR失敗として理由を表示する
- iCloud取得を許可している場合でも、通信状態や端末状態により取得に失敗することがある

## 注意表示

設定画面とOCR開始前の確認画面で次を表示する。

- iCloud写真を取得する場合は通信量が増える場合がある
- モバイル通信では時間と通信量に注意する
- iCloud取得は外部OCRサービス利用ではない
- 写真本体は変更しない

## 実機確認

1. オフライン優先にする
2. iCloud上のみの写真をOCR対象にして、失敗表示になっても落ちないことを確認する
3. iCloud取得を許可する
4. 同じ写真でOCRできるか確認する
5. モバイル通信環境では通信量に注意する
