# iOS ネイティブ版の将来計画

## 基本方針

iOSネイティブ版では SwiftUI と PhotoKit を使います。iCloud写真はAPIキーで直接読むのではなく、ユーザーが許可した写真ライブラリ権限を通して扱います。

## iCloud写真

- `PHPhotoLibrary` と PhotoKit を使って写真ライブラリを参照します。
- 端末とiCloud写真の同期状態は iOS の写真ライブラリ管理に任せます。
- 写真本体を外部送信しない設計を優先します。

## OneDrive

OneDrive は将来的に Microsoft ログインと Microsoft Graph で扱います。初期状態はOFFとし、ユーザーが明示的にONにした場合だけ連携します。

## OCR

OCR検索は Apple Vision framework など、端末内処理を優先します。外部API連携は任意設定とし、初期OFFにします。

## 秘密情報の保存

APIキー、アクセストークン、リフレッシュトークンを扱う場合は Keychain に保存します。平文ファイル保存は行いません。

## 外部送信しない設計

写真本体を外部送信しないことを基本設計にします。外部連携を追加する場合は、送信対象、送信先、保存期間、OFFにする方法を明示します。

## App Store 配布時の説明方針

App Store 配布時は、写真アクセス権限の用途、収集する情報、外部送信の有無、ユーザーが削除できるデータを明確に説明します。
