# しまい箱 デモ説明

## これは何か

しまい箱は、PCに同期済みの iCloud写真、OneDrive写真、任意フォルダの写真を、PCやスマホブラウザから探しやすくするローカル写真検索ツールです。

クラウド上の写真へ直接ログインせず、Windows上にある同期済みフォルダを読み取り専用でスキャンします。

## 何ができるか

- iCloud写真、OneDrive写真、任意フォルダをまとめて検索できます。
- ファイル名、フォルダ名、日付、拡張子、写真/動画、推定カテゴリで絞り込めます。
- サムネイルを見ながら探せます。
- OCRを開始すると、写真内の文字も検索できます。
- 重複候補を確認できます。
- PCだけで使えます。
- 明示的にLAN公開モードで起動した時だけ、同じ家庭内Wi-Fiのスマホから見られます。

## 安全方針

- 写真は外部送信しません。
- 読み取り専用です。
- 元写真を削除、移動、リネーム、上書きしません。
- 外部クラウドAPIや有料APIには接続しません。
- 実写真を試す場合は、まずコピーした小規模フォルダだけを対象にしてください。

## 起動方法

最初に一度だけセットアップします。

```powershell
.\Setup-DriveResearch.ps1
```

PCだけで使う場合は、次をダブルクリックします。

```text
Start-ShimaiBako.cmd
```

画面に表示されたPC用URLをブラウザで開きます。

## スマホで見る方法

スマホで見せる時だけ、次をダブルクリックします。

```text
Start-ShimaiBako-LAN.cmd
```

起動前に注意文が出ます。内容を確認してEnterを押すとLAN公開モードで起動します。

PowerShell画面に、スマホ用URLとPINが表示されます。iPhoneとPCを同じ家庭内Wi-Fiにつなぎ、Safariでスマホ用URLを開いてPINを入力してください。

公共Wi-Fi、会社Wi-Fi、ホテル、カフェ、共有オフィスでは使わないでください。ルーターのポート開放や外部トンネル公開もしないでください。

## PINの説明

LAN公開モードでは、同じWi-Fi内の第三者が勝手に検索画面やサムネイルへアクセスできないようにPINが必要です。

PINは起動のたびにランダム生成され、PowerShell画面にだけ表示されます。READMEや設定ファイルには固定PINを書きません。

## サンプル写真で試す方法

1. `Start-ShimaiBako.cmd` で起動します。
2. `スキャン` 画面でサンプルまたはテスト画像のデータソースをスキャンします。
3. `検索` 画面で `tokyo`、`Receipt`、`shimai`、`construction` などを検索します。
4. 検索結果を開き、サムネイル、元パス、撮影日、サイズ、OCR結果、推定カテゴリを確認します。

配布用デモでは、自作生成の `data/test_assets` 100枚を使う方針です。外部検証画像はライセンス確認が必要なため、標準デモZIPには同梱しません。

## 自分の写真で試す場合

元写真フォルダを直接登録せず、まず `data/real_test_photos` や任意の検証用フォルダに写真をコピーしてください。

最初は100〜300枚程度にして、スキャン最大件数とOCR最大件数も100〜300件にします。問題がないことを確認してから対象を広げてください。

## できないこと

- クラウド上だけにある写真は直接検索できません。
- iCloudやOneDriveへのログイン連携はまだありません。
- 写真の削除、移動、整理操作はできません。
- 動画サムネイルは環境によりプレースホルダーになります。
- OCR精度はTesseractと画像品質に依存します。

## 注意点

- LAN公開モードは家庭内Wi-Fiでの一時利用だけを想定しています。
- DBバックアップにはOCRテキストやファイルパスが含まれることがあります。
- デモ配布前には、配布前チェックを実行してください。

## よくあるトラブル

### 起動しない

`Setup-DriveResearch.ps1` を先に実行してください。PythonとNode.jsが必要です。

### スマホから開けない

`Start-ShimaiBako-LAN.cmd` で起動しているか、PCとスマホが同じ家庭内Wi-Fiにいるか、Windows FirewallでPythonやNode.jsがブロックされていないか確認してください。

### PINが分からない

LAN公開モードのPowerShell画面に表示されています。分からなくなった場合は一度終了し、起動し直してください。

### OCRできない

Tesseract OCR本体と、必要な言語データが必要です。統計画面でTesseractと `jpn` / `eng` の検出状態を確認してください。
